# Curated Glyph Library

A starter set of glyphs the skill can choose from, organized by category.
Mix emojis and Nerd Font icons — the user's font (a Nerd Font) renders both.

The skill should **select ~5 candidates** from the most relevant categories
for the predicted window context. Do not restrict yourself to this list if
a more fitting glyph exists (emoji or Nerd Font), but prefer these first.
